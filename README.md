# 🐰 淘气兔全自动签到与抵消脚本 (Taoqitu Checkin)

本项目包含两个版本的 Python 自动化脚本，专门用于「淘气兔 (Taoqitu)」平台的每日自动签到以及流量自动抵消。采用**动态账号密码登录**机制，完美绕过防护盾，彻底解决 Token 过期问题。

---

## 📂 版本说明

本项目提供两个独立的版本，你可以根据自己的需求选择配置其中一个，或同时保留：

1. **单账号极简版 (`sign.py`)**
   * 适合只有 1 个账号的用户。
   * 配置简单，即插即用。
2. **多账号循环版 (`sign_multi.py`)** 🌟 *推荐*
   * 适合拥有多个账号的用户。
   * 一次配置无限扩展，自动循环执行，内置随机延迟防风控机制。
   * 独立的流量监控面板，清晰掌握每个账号的签到剩余流量。

---

## 🚀 使用教程

### 第一步：Fork 仓库
点击右上角的 `Fork` 按钮，将本项目复制到你自己的 GitHub 账号下，并进入 **Actions** 标签页允许工作流运行。

### 第二步：配置账号变量 (Secrets)
在你的仓库中，依次点击 **Settings** -> **Secrets and variables** -> **Actions** -> **New repository secret**。根据你选择的版本添加变量：

#### 👉 若使用【单账号极简版】请添加：
* `USERNAME` : 你的登录邮箱账号
* `PASSWORD` : 你的登录密码

#### 👉 若使用【多账号循环版】请添加：
* `ACCOUNTS` : 多账号配置，**一行一个**，账号和密码用英文逗号隔开。示例：
  ```text
  user1@gmail.com,password123
  user2@qq.com,mypassword456
